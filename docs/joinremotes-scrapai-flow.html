JoinRemotes Admin Scraper Flow

Layman version: you paste a company job board URL in JoinRemotes, ScrapAI visits it, understands the jobs/company, then sends clean data back so JoinRemotes can create/update company profiles and job listings.

What the admin does

  • • Opens JoinRemotes Admin → Scraper
  • • Pastes a job board/careers URL
  • • Reviews the company/jobs ScrapAI found
  • • Approves import into JoinRemotes

What ScrapAI does

  • • Detects what kind of job board it is
  • • Finds company details and job pages
  • • Extracts messy web pages into structured JSON
  • • Sends results back by API/webhook

What JoinRemotes gets

  • • Company profiles ready to save
  • • Job listings ready to publish
  • • Source URLs for dedupe and updates
  • • A repeatable flow for refreshing listings
